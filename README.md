# Heart Disease Classification and Predictive Modeling

## Overview
This project analyzes a heart disease dataset and compares the performance of KNN, Decision Tree, and Logistic Regression classifiers.

## Objectives
- Identify predictors of heart disease
- Build classification models
- Compare model performance

## Methods
- Exploratory Data Analysis (EDA)
- Missing value handling
- Feature selection
- 5-fold cross-validation
- ROC/AUC evaluation

## Results

| Model | AUC |
|---------|---------|
| KNN | 0.957 |
| Logistic Regression | 0.929 |
| Decision Tree | 0.895 |
